# Descripción de columnas: track_data_final.csv

| Columna               | Descripción                                                                 |
|-----------------------|-----------------------------------------------------------------------------|
| track_id              | ID único de la canción en Spotify                                            |
| track_name            | Nombre de la canción                                                         |
| track_number          | Número de pista dentro del álbum                                             |
| track_popularity      | Popularidad de la canción (0-100, según Spotify)                             |
| track_duration_ms     | Duración de la canción en milisegundos                                      |
| explicit              | Indica si la canción es explícita (True/False)                              |
| artist_name           | Nombre del artista principal                                                 |
| artist_popularity     | Popularidad del artista (0-100, según Spotify)                               |
| artist_followers      | Número de seguidores del artista en Spotify                                  |
| artist_genres         | Lista de géneros asociados al artista                                        |
| album_id              | ID único del álbum en Spotify                                                |
| album_name            | Nombre del álbum                                                             |
| album_release_date    | Fecha de lanzamiento del álbum (AAAA-MM-DD)                                  |
| album_total_tracks    | Número total de pistas en el álbum                                           |
| album_type            | Tipo de álbum (por ejemplo: album, single, compilation)                      |
